首先，我们要知道什么是二叉搜索树，一开始我也忘了，百度才想起来是啥。简单来说就是右子树所有节点的值大于根节点，左子树所有节点的值小于根节点。题目要求我们将其按要求转换为累加树，累加比该节点的值大的节点，显然，先遍历右子树再遍历左子树，即反序中序遍历。

---



下面是代码：

~~~C++
class Solution {
public:
    int sum = 0;    //记录累加和
    TreeNode* convertBST(TreeNode* root) {
        if(root){
            convertBST(root->right);
            sum += root->val;
            root->val = sum;
            convertBST(root->left);
        }
        return root;
    }
};
~~~
